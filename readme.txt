== Description ==

Displays a list of Instagram images from a specific user using a Wordpress shortcode.

## Usage:
`[getinsta howmany="HOWMANYIMAGES" userid="YOURUSERID" client_id="YOURCLIENTID"]`


== Installation ==


1. Upload `getinsta` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get a client_id from [Instagram](http://instagram.com/developer/)
4. Place `[getinsta howmany="HOWMANYIMAGES" userid="YOURUSERID" client_id="YOURCLIENTID"]` in your page or post.
