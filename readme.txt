== Description ==

Displays a list of Instagram images from a specific user using a shortcode.

*Usage:*
´[getinsta howmany="HOWMANYIMAGES" userid="YOURUSERID" client_id="YOURCLIENTID"]´


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `getinsta` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[getinsta howmany="HOWMANYIMAGES" userid="YOURUSERID" client_id="YOURCLIENTID"]` in your page or post.
